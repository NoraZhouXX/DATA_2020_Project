4 Part 1: Paired t-test

4.1 Research Question

Do students show a statistically significant improvement from pre-test to post-test scores?

4.2 Question 1

Clearly state the null and alternative hypotheses.

4.3 Answer

H0: The mean difference between post-test and pre-test scores is equal to 0.
HA: The mean difference between post-test and pre-test scores is greater than 0.

4.4 Question 2

Explain why a paired t-test is appropriate in this setting.

4.5 Answer

A paired t-test is appropriate because the pre-test and post-test scores are measured on the same individuals. This creates dependent observations, and the test accounts for within-subject variation.

4.6 Question 3

Conduct the paired t-test.

paired_test <- t.test(lab_data$post_score, lab_data$pre_score,
                     paired = TRUE, alternative = "greater")

4.7 Question 4

Interpret the p-value in the context of this problem.

4.8 Answer

If the p-value is less than 0.05, we reject the null hypothesis and conclude that there is statistically significant evidence that students improved after the intervention.

4.9 Question 5

Compute and interpret the confidence interval for the mean difference.

[1] 3.820046      Inf
attr(,"conf.level")
[1] 0.95

4.10 Answer

If the confidence interval lies entirely above 0, this indicates that the average improvement is positive, reinforcing the conclusion that students improved.

4.11 Question 6

Explain what a Type I error and a Type II error would mean in this context.

4.12 Answer

A Type I error would occur if we conclude that students improved when, in reality, there was no improvement.
A Type II error would occur if we fail to detect an actual improvement in student performance.

8 Part 5: Confidence Intervals

8.1 Question

Explain the relationship between hypothesis testing and confidence intervals.

8.2 Answer

Confidence intervals provide a range of plausible values for a parameter. If the null hypothesis value lies outside the confidence interval, we reject the null hypothesis. If it lies within the interval, we fail to reject the null hypothesis.

10 Part 7: Assumptions

10.1 Question

List and explain the key assumptions required for t-tests.

10.2 Answer

  • Independence: Observations must be independent of each other.
  • Normality: The data should be approximately normally distributed, or the sample size should be sufficiently large.
  • Equal variance (for two-sample tests): The variability in both groups should be similar, depending on the test used.

11 Part 8: Effect Size vs Statistical Significance

11.1 Question

Explain the difference between statistical significance and practical significance.

11.2 Answer

Statistical significance indicates whether an observed effect is unlikely to be due to chance. Practical significance considers whether the magnitude of the effect is meaningful in a real-world context. A result can be statistically significant but still have little practical importance.

12 Final Reflection

12.1 Question

What is the most important takeaway from hypothesis testing?

12.2 Answer

Hypothesis testing provides a structured framework for making decisions based on data, but results must always be interpreted carefully in context, considering assumptions, effect sizes, and real-world implications.
